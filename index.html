Articulos.

HTML

HTML, siglas en inglés de HyperText Markup Language referencia

al lenguaje de marcado para la elaboración de páginas web.

HISTORIA

El origen de HTML se remonta a 1980, cuando el físico

Tim Berners-Lee, investigador del CERN (Organización

Europea para la Investigación Nuclear) propuso un nuevo

sistema de “hipertexto” para compartir documentos. Los

sistemas de hipertexto habían sido desarrollados años

antes.

ETIQUETAS DEL HTML.

Las etiquetas HTML son fragmentos de código que nos permiten crear elementos HTML.

Los elementos son la estructura básica de HTML.

Dichos elementos tienen dos propiedades básicas: atributos y contenido.
